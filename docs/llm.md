# Full Disclosure:

Yes, I used LLMs to build and maintain MinRender. I built it over months as a side project for internal use and currently use it to manage a facility's render farm. I am only sharing it because there might be an idea or two in there that others can benefit from. The phonebook-based mesh is particularly unique, negating the need for static IPs or any networking know-how; the leadership election keeps the renders up until the last node dies.
