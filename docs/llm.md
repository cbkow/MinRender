# Full Disclosure:

Yes, I used LLMs to build and maintain MinRender. I built it over months as a side project for internal use and currently use it to manage a facility's render farm. (I didn't expect it to work out; I started it as a curiosity). I am only sharing it because there might be an idea or two in there that others can benefit from. The phonebook-based mesh is particularly interesting and unique, setting it apart from existing services. The leadership election will keep the renders going up until the last node dies.
